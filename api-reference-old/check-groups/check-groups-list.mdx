## Overview

The V2 List Check Groups endpoint provides an enhanced version of check group listing with improved performance, additional metadata, and extended filtering capabilities compared to the V1 API.

**Common Use Cases**:
- Enhanced Group Management
- Advanced Filtering
- Performance Optimization
- Extended Metadata Access

<Note>
The V2 API provides enhanced features and improved performance. Use V2 endpoints for new integrations while maintaining backward compatibility with V1 for existing implementations.
</Note>
