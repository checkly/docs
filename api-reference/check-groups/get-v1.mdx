## Overview

The Get Check Group endpoint retrieves comprehensive details about a specific check group, including its configuration, member checks, and group-level settings.

**Common Use Cases**:
- Group Configuration Review
- Member Check Inspection
- Group Settings Verification
- Group Management

<Note>
This endpoint returns complete group details including all member checks, shared settings, and group-level configurations.
</Note>
