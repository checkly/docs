## Overview

The Multistep Check Analytics endpoint provides comprehensive metrics and aggregated data for multistep checks across custom time ranges. Use this endpoint to analyze the performance of your multi-step API workflows and transaction monitoring.

**Common Use Cases**:
- Multi-Step Transaction Analysis
- Workflow Performance Monitoring
- API Chain Reliability Metrics
- Response Time Analysis

**Rate Limiting**: 30 requests per 60 seconds

<Note>
This endpoint supports various aggregation methods and filtering options to help you analyze complex multi-step check performance across different time periods and geographic locations.
</Note>
