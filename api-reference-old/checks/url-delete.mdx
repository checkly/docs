## Overview

The Delete URL Check endpoint permanently removes a URL monitoring check from your account. This action cannot be undone and will stop all website monitoring for the deleted check.

**Common Use Cases**:
- Check Cleanup
- Website Monitoring Decommissioning
- Resource Management
- URL Retirement

<Warning>
Deleting URL checks is permanent and cannot be undone. All historical response time data and availability metrics will be permanently removed.
</Warning>

<Note>
Consider deactivating checks instead of deleting them to preserve historical website monitoring data and performance trends.
</Note>
