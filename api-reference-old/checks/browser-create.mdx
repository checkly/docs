## Overview

The Create Browser Check endpoint allows you to create new browser-based monitoring checks using Playwright scripts to test user journeys, page performance, and web application functionality.

**Common Use Cases**:
- User Journey Monitoring
- E-commerce Flow Testing
- Page Performance Monitoring
- Web Application Health Checks

<Note>
Browser checks support Playwright scripts with full browser automation capabilities. Configure appropriate timeouts and resource limits for reliable execution.
</Note>
