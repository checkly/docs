## Overview

The List Static IPs endpoint returns all static IP addresses used by Checkly's monitoring infrastructure. Use this endpoint to configure firewall rules, allowlists, and security policies for your monitored services.

**Common Use Cases**:
- Firewall Configuration
- Security Allowlisting
- Network Access Control
- IP-based Filtering

<Note>
Static IP addresses may change periodically. Subscribe to Checkly updates or check this endpoint regularly to maintain accurate allowlists and security configurations.
</Note>
