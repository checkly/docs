## Overview

The Update Client Certificate endpoint allows you to update existing client certificates, including certificate renewal, metadata updates, and configuration changes.

**Common Use Cases**:
- Certificate Renewal
- Metadata Updates
- Configuration Changes
- Certificate Rotation

<Warning>
Updating certificates may temporarily affect monitoring for checks using the certificate. Coordinate updates to minimize monitoring disruptions.
</Warning>

<Note>
Certificate updates take effect immediately for new check executions. Ensure new certificates are valid and properly configured before deployment.
</Note>
