## Overview

The List Static IPv6 Addresses endpoint returns all static IPv6 addresses used by Checkly's monitoring infrastructure. Use this endpoint to configure IPv6-specific firewall rules and network policies.

**Common Use Cases**:
- IPv6 Firewall Configuration
- Network Security Policies
- Dual-Stack Network Setup
- IPv6 Access Control

<Note>
Configure both IPv4 and IPv6 allowlists for comprehensive network access control. IPv6 addresses may be used depending on your network configuration and Checkly's routing decisions.
</Note>
