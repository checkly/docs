## Overview

The Static IPs Text Format endpoint returns all Checkly static IP addresses as a plain text list, with one IP address per line. This format is ideal for importing directly into firewall configurations and automated scripts.

**Common Use Cases**:
- Automated Firewall Configuration
- Script Integration
- Bulk IP Import
- Configuration File Generation

**Response Format**: Plain text with one IP address per line

<Note>
This endpoint returns the same IP addresses as the JSON endpoint but in a plain text format that's easier to process with shell scripts and configuration management tools.
</Note>
