## Overview

The List Alert Notifications endpoint returns all alert notifications that have been sent from your account. Use this endpoint to track notification delivery, troubleshoot notification issues, and audit alert activity.

**Common Use Cases**:
- Notification Delivery Tracking
- Alert Audit Trails
- Troubleshooting Missed Alerts
- Notification Performance Analysis

**Rate Limiting**: 5 requests per 10 seconds

**Time Range**: Returns data within a 24-hour timeframe

<Note>
This endpoint provides detailed information about notification delivery including timestamps, success/failure status, and delivery channels. Use time range parameters to filter results.
</Note>
