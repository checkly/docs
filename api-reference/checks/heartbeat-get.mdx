## Overview

The Get Heartbeat Check endpoint retrieves comprehensive details about a specific heartbeat monitoring check, including expected intervals, grace periods, and signal configuration.

**Common Use Cases**:
- Heartbeat Check Configuration Review
- Interval Setting Inspection
- Grace Period Verification
- Check Management

<Note>
This endpoint returns complete heartbeat check configuration including expected ping intervals, grace periods, and signal tracking settings.
</Note>
