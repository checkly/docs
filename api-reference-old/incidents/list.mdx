## Overview

The List Incidents endpoint returns all incidents that have been created in your Checkly account. Incidents track service disruptions, outages, and other operational events.

**Common Use Cases**:
- Incident History Review
- Operational Reporting
- Incident Management
- Service Reliability Analysis

<Note>
Incidents can be automatically created from check failures or manually created for operational events. Use this endpoint to track and manage service disruptions.
</Note>
