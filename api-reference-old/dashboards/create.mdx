## Overview

The Create Dashboard endpoint allows you to create new monitoring dashboards for displaying check results, performance metrics, and status information. Configure dashboard layout, check assignments, and access settings.

**Common Use Cases**:
- Team Dashboard Creation
- Public Status Displays
- Executive Reporting
- Custom Monitoring Views

<Note>
Dashboards can be configured for public access or restricted to team members. Choose appropriate visibility settings based on your use case.
</Note>
