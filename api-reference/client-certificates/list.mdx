## Overview

The List Client Certificates endpoint returns all client certificates configured in your Checkly account. Client certificates enable mutual TLS authentication for monitoring services that require certificate-based authentication.

**Common Use Cases**:
- Certificate Management
- mTLS Configuration
- Security Auditing
- Certificate Inventory

<Note>
Client certificates are used for mutual TLS authentication when monitoring services that require certificate-based client authentication. Manage certificates securely and monitor expiration dates.
</Note>
