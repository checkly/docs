## Overview

The Get TCP Check endpoint retrieves comprehensive details about a specific TCP monitoring check, including connection parameters, timeout settings, and monitoring configuration.

**Common Use Cases**:
- TCP Check Configuration Review
- Connection Parameter Inspection
- Timeout Setting Verification
- Check Management

<Note>
This endpoint returns complete TCP check configuration including host, port, connection timeouts, and retry settings.
</Note>
