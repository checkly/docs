## Overview

The List Private Location Keys endpoint returns all API keys associated with a specific private location. These keys are used by monitoring agents to authenticate and report check results.

**Common Use Cases**:
- Key Management
- Agent Configuration
- Security Auditing
- Access Control

<Note>
API keys are required for private location agents to authenticate with Checkly. Manage keys securely and rotate them regularly for optimal security.
</Note>
