## Overview

The Update API Check endpoint allows you to modify existing API monitoring checks, including request parameters, response assertions, scheduling, and monitoring locations.

**Common Use Cases**:
- API Check Maintenance
- Request Parameter Updates
- Assertion Modifications
- Schedule Changes

<Warning>
Updating API checks may temporarily affect monitoring. Ensure changes are tested and consider the impact on alerting and reporting.
</Warning>

<Note>
Changes to API checks take effect immediately. Test your updates in a non-production environment when possible.
</Note>
