## Overview

The Update Heartbeat Check endpoint allows you to modify existing heartbeat monitoring checks, including expected intervals, grace periods, scheduling settings, and notification preferences.

**Common Use Cases**:
- Interval Adjustments
- Grace Period Updates
- Schedule Changes
- Notification Updates

<Warning>
Updating heartbeat checks may affect process monitoring. Ensure interval and grace period changes match your application's actual behavior.
</Warning>

<Note>
Changes to heartbeat checks take effect immediately. Coordinate updates with your application's ping schedule to avoid false alerts.
</Note>
