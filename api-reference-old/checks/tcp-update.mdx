## Overview

The Update TCP Check endpoint allows you to modify existing TCP monitoring checks, including connection parameters, timeout settings, scheduling, and monitoring locations.

**Common Use Cases**:
- Connection Parameter Updates
- Timeout Adjustments
- Schedule Modifications
- Location Changes

<Warning>
Updating TCP checks may temporarily affect network monitoring. Ensure changes are appropriate for your network infrastructure.
</Warning>

<Note>
Changes to TCP checks take effect immediately. Test connectivity before updating production network monitoring.
</Note>
