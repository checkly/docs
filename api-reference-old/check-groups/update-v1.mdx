## Overview

The Update Check Group endpoint allows you to modify existing [check groups](/platform/groups).

<Warning>
  Groups updated with this endpoint will always override its member check settings (i.e. location, alerting, etc.). We recommend using the [V2 endpoint](/api-reference/check-groups/update/) instead for more flexibility.
</Warning>
