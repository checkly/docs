## Overview

The List Static IPs by Region endpoint returns all static IP addresses used by Checkly's monitoring infrastructure, organized by geographic region. This endpoint helps you configure region-specific firewall rules and network access policies.

**Common Use Cases**:
- Regional Firewall Configuration
- Geographic IP Allowlisting
- Region-Specific Security Policies
- Network Access Control by Location

<Note>
Use this endpoint when you need to allowlist Checkly IPs for specific geographic regions or configure region-based network security policies.
</Note>
