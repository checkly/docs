## Overview

The V2 Get Check Results endpoint provides enhanced check result data with additional metrics, improved performance, and extended filtering capabilities compared to the V1 API.

**Common Use Cases**:
- Enhanced Result Analysis
- Extended Metrics Access
- Performance Optimization
- Advanced Filtering

<Note>
V2 check results include additional performance metrics, enhanced error details, and improved data structures for better analysis and reporting.
</Note>
