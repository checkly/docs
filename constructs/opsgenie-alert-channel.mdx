Sends an alert notification to your Opsgenie account.

<CodeGroup>

```ts Basic Example
import { OpsgenieAlertChannel } from 'checkly/constructs'

const opsGenieChannel = new OpsgenieAlertChannel('opsgenie-channel-1', {
  name: 'My Ops Team',
  region: 'EU',
  priority: 'P1',
  apiKey: 'xxxx123abc'
})
```

```ts Advanced Example
import { OpsgenieAlertChannel } from 'checkly/constructs'

const opsGenieChannel = new OpsgenieAlertChannel('opsgenie-channel-1', {
  name: 'My Ops Team',
  region: 'EU',
  priority: 'P1',
  apiKey: 'xxxx123abc',
  sendRecovery: true,
  sendFailure: true,
  sendDegraded: true,
  sslExpiry: true,
  sslExpiryThreshold: 30,
})
```

</CodeGroup>

<Note>
  If you need to reference existing alert channels that were created outside of your CLI project, use [`fromId()`](/constructs/alert-channel#using-fromid-to-reference-an-existing-channel).
</Note>

## Configuration

### Opsgenie Alert Channel Options

<ResponseField name="name" type="string" required>
Friendly name to recognise the integration.
</ResponseField>

<ResponseField name="region" type="string" required>
A string representing the Opsgenie location, either `EU` or `US`.
</ResponseField>

<ResponseField name="priority" type="string" required>
A string representing the severity level, `P1` to `P5`.
</ResponseField>

<ResponseField name="apiKey" type="string" required>
An API key for your Opsgenie account.
</ResponseField>
