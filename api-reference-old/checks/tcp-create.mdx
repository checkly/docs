## Overview

The Create TCP Check endpoint allows you to create new TCP port monitoring checks to verify network connectivity, port availability, and service responsiveness at the network level.

**Common Use Cases**:
- Port Availability Monitoring
- Network Service Health Checks
- Database Connection Monitoring
- Service Discovery Verification

<Note>
TCP checks test network-level connectivity and port availability. Configure appropriate timeouts and retry settings for reliable network monitoring.
</Note>
