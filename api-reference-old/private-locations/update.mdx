## Overview

The Update Private Location endpoint allows you to modify existing private locations, including name, description, and configuration settings for your monitoring agents.

**Common Use Cases**:
- Location Name Updates
- Configuration Changes
- Description Updates
- Settings Modifications

<Note>
Changes to private locations take effect immediately but may require agent restart for some configuration updates. Coordinate changes with your infrastructure team.
</Note>
