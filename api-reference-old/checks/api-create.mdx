## Overview

The Create API Check endpoint allows you to create new API monitoring checks to monitor REST APIs, GraphQL endpoints, and other HTTP-based services. Configure request parameters, assertions, and monitoring settings.

**Common Use Cases**:
- API Endpoint Monitoring
- REST API Health Checks
- GraphQL Query Monitoring
- API Performance Tracking

<Note>
API checks support various HTTP methods, custom headers, request bodies, and response assertions. Configure appropriate timeouts and retry settings for reliable monitoring.
</Note>
