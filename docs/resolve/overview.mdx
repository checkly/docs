## What is Checkly Resolve?

Checkly Resolve provides advanced debugging and incident resolution capabilities through distributed tracing and AI-powered analysis. When issues are detected, Resolve helps you quickly understand the root cause and get back to normal operations.

## How It Works

<Steps>
<Step title="Instrument Your Application">
Add tracing to your application stack:
- Install OpenTelemetry instrumentation
- Configure trace collection and export
- Set up automatic instrumentation for popular frameworks
</Step>

<Step title="Import Trace Data">
Send trace data to Checkly:
- Import via OpenTelemetry standard protocols
- Use HTTPS or gRPC endpoints for trace ingestion
- Configure sampling and filtering for optimal performance
</Step>

<Step title="Analyze Issues with AI">
Leverage Rocky AI for incident resolution:
- Get automated root cause analysis
- Receive intelligent recommendations for fixes
- Access AI-powered insights during incidents
</Step>

<Step title="Export and Integrate">
Connect with your existing observability stack:
- Export traces to Grafana Tempo, Honeycomb, New Relic
- Integrate with your incident response workflows
- Share insights with development and operations teams
</Step>
</Steps>
