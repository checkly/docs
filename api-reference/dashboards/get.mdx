## Overview

The Get Dashboard endpoint retrieves comprehensive details about a specific dashboard, including its configuration, displayed checks, layout settings, and access permissions.

**Common Use Cases**:
- Dashboard Configuration Review
- Settings Inspection
- Access Control Verification
- Dashboard Management

<Note>
This endpoint returns complete dashboard configuration including check assignments, layout preferences, and sharing settings.
</Note>
