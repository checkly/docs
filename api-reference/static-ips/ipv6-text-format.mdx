## Overview

The Static IPv6 Addresses Text Format endpoint returns all Checkly static IPv6 addresses as a plain text list, with one IPv6 address per line. This format is ideal for importing directly into firewall configurations and automated scripts.

**Common Use Cases**:
- Automated IPv6 Firewall Configuration
- Script Integration
- Bulk IPv6 Import
- Configuration File Generation

**Response Format**: Plain text with one IPv6 address per line

<Note>
This endpoint provides the same IPv6 addresses as the JSON endpoint but in a plain text format optimized for automation and configuration management tools.
</Note>
