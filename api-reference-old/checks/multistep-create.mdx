## Overview

The Create Multistep Check endpoint allows you to create new multistep API monitoring checks to test complex API workflows, transaction flows, and multi-endpoint dependencies.

**Common Use Cases**:
- API Workflow Testing
- Transaction Flow Monitoring
- Multi-Endpoint Dependencies
- Complex API Scenarios

<Note>
Multistep checks allow you to chain multiple API requests together, passing data between steps to test complete user journeys and business processes.
</Note>
