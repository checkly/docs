## Overview

The Get Browser Check endpoint retrieves comprehensive details about a specific browser monitoring check, including Playwright scripts, execution settings, and performance configurations.

**Common Use Cases**:
- Browser Check Configuration Review
- Script Content Inspection
- Performance Setting Verification
- Check Management

<Note>
This endpoint returns complete browser check configuration including Playwright scripts, environment variables, and execution parameters.
</Note>
