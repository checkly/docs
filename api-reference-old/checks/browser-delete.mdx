## Overview

The Delete Browser Check endpoint permanently removes a browser monitoring check from your account. This action cannot be undone and will stop all monitoring for the deleted check.

**Common Use Cases**:
- Check Cleanup
- User Journey Decommissioning
- Resource Management
- Application Retirement

<Warning>
Deleting browser checks is permanent and cannot be undone. All historical data, screenshots, and performance metrics will be permanently removed.
</Warning>

<Note>
Consider deactivating checks instead of deleting them to preserve historical data and performance trends.
</Note>
