## Overview

The Get Private Location Metrics endpoint provides performance and health metrics for a specific private location, including agent status, check execution statistics, and connectivity information.

**Common Use Cases**:
- Agent Performance Monitoring
- Health Status Verification
- Capacity Planning
- Troubleshooting

<Note>
Private location metrics help monitor agent performance, connectivity, and resource utilization to ensure reliable monitoring from your infrastructure.
</Note>
