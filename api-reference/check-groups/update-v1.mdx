## Overview

The Update Check Group endpoint allows you to modify existing check groups, including group settings, shared configurations, and group-level properties that apply to all member checks.

**Common Use Cases**:
- Group Configuration Updates
- Shared Settings Modification
- Group Name Changes
- Environment Updates

<Warning>
Updating group settings affects all member checks. Ensure changes are appropriate for all checks in the group.
</Warning>

<Note>
Changes to check groups apply to all member checks immediately. Consider the impact on active monitoring when updating group configurations.
</Note>
