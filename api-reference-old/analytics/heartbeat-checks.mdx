## Overview

The Heartbeat Check Analytics endpoint provides detailed availability metrics and aggregated data for heartbeat checks across custom time ranges. This endpoint allows you to analyze the performance and reliability of your heartbeat monitoring.

**Common Use Cases**:
- Heartbeat Performance Analysis
- Availability Reporting
- Service Reliability Metrics
- Historical Trend Analysis

**Rate Limiting**: 600 requests per 60 seconds

<Note>
This endpoint provides specialized metrics for heartbeat checks, including availability percentages and retry statistics across configurable time ranges and aggregation intervals.
</Note>
