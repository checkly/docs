## Overview

The Get Private Location endpoint retrieves comprehensive details about a specific private location, including configuration, status, agent information, and connectivity details.

**Common Use Cases**:
- Private Location Configuration Review
- Agent Status Monitoring
- Connectivity Verification
- Location Management

<Note>
This endpoint returns complete private location details including agent status, network configuration, and monitoring capabilities.
</Note>
